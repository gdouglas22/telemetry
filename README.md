# Smart Home

Учебный многомодульный проект системы умного дома.

## Структура

- `telemetry` — сервисы обработки телеметрических данных
  - `collector` — приём событий датчиков и хабов по HTTP (JSON) и публикация в Kafka в формате Avro
  - `serialization`
    - `avro-schemas` — Avro-схемы событий и сгенерированные классы
- `infra` — инфраструктурные компоненты (будет добавлено позже)
- `commerce` — сервисы продаж устройств (будет добавлено позже)

## Сборка

```
mvn clean install
```
